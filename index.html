The gsap object has 3 main methods for creating Tweens and optionally adding them to Timelines:

gsap.to()
gsap.from()
gsap.fromTo()

What is a Tween?

A Tween can change a single property of a single object over time.

A Tween can change a multiple properties of a single object over time.

A Tween can change a multiple properties of multiple objects over time.

A Tween can multiple objects with staggered start times.

from and fromTo tweens

gsap.from animates from the values you specify to the object’s natural values.

gsap.fromTo animates from the values you specify to the values you specify.
